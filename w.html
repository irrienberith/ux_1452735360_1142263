OEM接口说明

OEM接口类 IFOEMUtils

1.服务器列表

  • 显示服务器列表 > void showServerOptions()

  • 添加服务器信息

    void addServerWithName(final Context context, final String serverName, final String serverUrl, final boolean userEditable)

    参数列表:android上下文,服务器名,服务器地址,是否可被用户编辑 新添加的服务器名与已有服务器重名，或服务器地址不是合法的711以上服务器地址，都会导致添加失败。下同。

  • 添加服务器信息和用户信息

    • 添加的同时指定服务器是否可被用户编辑 > void addServerWithName(Context context, String serverName, String serverUrl, String userName, String password, boolean userEditable)

    参数列表:android上下文,服务器名,服务器地址,用户名，密码，是否可被用户编辑

    • 添加的同时默认服务器可被用户编辑 > void addServerWithName(Context context, String serverName, String serverUrl, String userName, String password)

    参数列表:android上下文,服务器名,服务器地址,用户名，密码

  • 删除服务器

    • 删除指定名称和地址的服务器 > void removeServer(Context context, String serverName, String serverUrl)

    参数列表:android上下文,服务器名,服务器地址

    • 删除指定名称的服务器 > void removeServer(Context context, String serverName)

    参数列表：android上下文,服务器名

    • 删除DEMO服务器 > void removeDemoServer(Context context)

    参数列表:android上下文

2.关于界面

  • 设置关于界面图标 > void setAppIcon(String appIcon)

参数:图片文件名。例如想把/drawable/xxx.png作为图标，则参数为:xxx

  • 设置关于界面app名称* > void setAppName(String appName)

参数:app名称

  • 设置版本号 > void setAppVersion(String version)

参数:版本号

  • 设置版本日期 > void setAppDate(String date)

参数:版本日期

  • 设置版权信息 > void setCopyrightInfo(String info)

参数:版权信息

  • 移除FR关于界面 > void removeAbout()

3.自动更新

  • 关闭FR自动更新 > void disableUpdate()

  • 打开FR自动更新 > void enableUpdate()
